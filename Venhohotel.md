## 9. HƯỚNG DẪN VẬN HÀNH WEBSITE

### Chạy local
```bash
cd "E:\Claude-Workspace\projects\Ven Ho Hotel"
npm install
npm run dev
# Mở http://localhost:3000
```

### Build & deploy
```bash
npm run build   # Build production
npx serve out   # Xem static build
```

### Cập nhật nội dung thường xuyên
Mở file `src/lib/data/content.ts`, tìm phần `westLake.body` và chỉnh:
```ts
body: "Nội dung bài viết mới về trải nghiệm Hồ Tây..."
```

### Thêm ảnh mới
1. Copy ảnh vào đúng thư mục: `public/images/[Tên-folder]/`
2. Cập nhật đường dẫn trong `src/lib/data/rooms.ts`
3. Chạy lại `npm run build`
